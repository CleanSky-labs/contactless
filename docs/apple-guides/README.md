# Apple Guides Bundle

This directory consolidates the Apple UX material gathered during the project review, focused on applying practical HIG patterns to the Settings screen.

## Included local documents

- `sources/guia-apple-ux.md`
  - Long-form strategic synthesis used as research input.
- `sources/APPLE_UI_LINES_MAESTRAS.md`
  - Distilled actionable guideline set for implementation decisions.

## Official Apple references used

The current Apple HIG site is heavily JavaScript-rendered, so direct static scraping is limited in this environment.
These official URLs were used as primary references and are preserved in `official-links.md`.
