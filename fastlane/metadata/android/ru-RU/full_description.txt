CleanSky — приложение для бесконтактных платежей криптовалютой через NFC. Просто поднесите телефон.

ВАШИ КЛЮЧИ, ВАШИ СРЕДСТВА, ВАША НЕЗАВИСИМОСТЬ

• Кошелёк с самохранением — вы контролируете свои приватные ключи
• Без регистрации — без email, без телефона, без KYC
• Работает офлайн — подписывайте транзакции без интернета
• Открытый код — проверяйте, не доверяйте

ПРИВАТНОСТЬ В ОСНОВЕ

• Скрытые адреса (EIP-5564) — каждый платёж идёт на уникальный адрес
• Приватность плательщика (ERC-4337) — скрывайте адрес кошелька при оплате
• Без отслеживания, без аналитики, без сбора данных

ПРОСТО И БЫСТРО

• Коснитесь для оплаты — NFC-платежи за секунды
• Коснитесь для приёма — принимайте платежи как продавец
• Мульти-чейн — Ethereum, Polygon, Arbitrum, Base и другие
• Без газа — платите без ETH на комиссии

ДЛЯ РЕАЛЬНОГО ИСПОЛЬЗОВАНИЯ

• Работает с нестабильными валютами — защитите сбережения
• Поддержка 70+ языков
• Для простых телефонов — работает на Android 7+
• Не требует Google Services

БЕЗОПАСНОСТЬ

• Шифрование AES-256-GCM с Android Keystore
• Биометрическая аутентификация (отпечаток/лицо)
• Поддержка StrongBox HSM на совместимых устройствах
• Без облачного бэкапа — ключи никогда не покидают устройство
